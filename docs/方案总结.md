基于大语言模型训练推荐模型
基座模型是Instruct model。推荐模型相当于微调。
把 item id 转换为 sids，快手论文是3个 sid。
sid 相当于文本，加入到大语言模型的 tokenizer。由tokenizer分配新的token ID。
构建用户交互序列样本。形式是 list sids。 做 ntp loss 训练。预测下一个交互 item 的 token id。
tokenizer 将预测的 token id 转成 sids。 再由维护转换表，将 sids 转换成 item id。 评估指标是 hit rate 和 ndcg。


如果不使用大语言模型的Instruct model，sid 直接就是 token id。 这才是 one-rec 方案
